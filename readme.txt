Easy Link Tracker is a link tracking management tool that allows you to create, manage and track internal & outbound links on your website.

== Description ==

Easy Link Tracker is a complete link tracking management tool that gives you the ability to create, manage and track internal & outbound links on your site by using custom post types, 301 redirects and Event Tracking using Google Analytics.


== Installation ==

1. Activate the plugin
2. Navigate to the 'Easy Link' menu. 
3. Create a new URL, or manage existing URLs.
4. Publish and use the URLs however you want!

== Frequently Asked Questions ==

= Getting a 404 error? =

To get rid of the 404 error when viewing a hyperlink, you need to resave your permalink structure. Go to Settings > Permalinks and click "Save Changes".

= Can I change the URL structure to use something other than /go/ ??? = 

Not without modifying the source code. We're working on a premium version that allows custom slugs and more.

= Will uninstalling the plugin remove all of my Easy Links? =

Yes! If you click uninstall, this plugin removes all traces of its existence.

= Why are the links generated using the ezLink shortcode nofollowed =

According to Google, you need to make sure that any paid links on your site don't pass PageRank. To do so, you need to make sure that any paid hyperlinks have the rel="nofollow" attribute.
